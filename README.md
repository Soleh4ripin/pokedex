# Pokédex App

Aplikasi Pokédex berbasis **NestJS** + **Handlebars**.

---

## 🚀 Fitur

- 🔍 Lihat daftar dan detail Pokémon  
- 🎯 Tangkap dan lepaskan Pokémon  
- 📦 Simpan koleksi pribadi  
- 🕓 Riwayat pelepasan  

---

## ⚙️ Teknologi

- 🧠 NestJS (Backend)
- 🍃 MongoDB (Database)
- 🎨 Handlebars + Bootstrap (Frontend)
